# 10 нестандартных сеток и систем компоновки
*Только приёмы сложности ⭐⭐ и ⭐⭐⭐ — без банальщины*

---

## 1. Золотое сечение как система колонок ⭐⭐⭐

**Суть:** Вся страница делится по φ = 1.618. Основной контент занимает большую часть, второстепенный — меньшую. Пропорция повторяется рекурсивно внутри каждого блока.

**Как реализовать:**
- Возьми ширину контейнера, например 1200px
- Основной блок: 1200 / 1.618 ≈ 741px
- Боковой блок: 1200 − 741 = 459px
- Внутри основного блока снова делишь по φ — и так вглубь

**Где работает:** лендинги, редакционные страницы, портфолио, журнальные макеты

**Эффект:** ощущение «правильности» без объяснения причины. Глаз не устаёт.

---

## 2. Sequence Grid на базе Фибоначчи ⭐⭐

**Суть:** Ряд 3 — 5 — 8 — 13 — 21 — 34 задаёт все размеры в системе. Ты выбираешь одну единицу (например, 1 = 8px) и умножаешь.

**Как реализовать:**
```
xs-отступ:   3 × 8 = 24px
sm-отступ:   5 × 8 = 40px
md-отступ:   8 × 8 = 64px
lg-отступ:  13 × 8 = 104px
xl-блок:    21 × 8 = 168px
заголовок:  34 × 8 = 272px (font-size или высота hero-зоны)
```

**Где работает:** система спейсингов в design system, типографические шкалы, размеры карточек

**Эффект:** любые два элемента визуально «родственны» — даже если не понятно почему

---

## 3. Broken Grid — намеренный выход за пределы ⭐⭐

**Суть:** Строгая сетка задаётся, а 1–2 элемента намеренно её нарушают — картинка заезжает за колонку, заголовок залезает в поле отступа, блок пересекает горизонтальную границу секции.

**Как реализовать:**
```css
.hero-image {
  margin-left: -80px;      /* выезжает за левую границу */
  width: calc(100% + 80px);
}

.accent-title {
  position: relative;
  left: -40px;             /* заголовок в поле отступа */
}
```

**Где работает:** fashion, агентства, портфолио, creative studios

**Эффект:** напряжение, движение, характер — страница перестаёт быть «шаблонной»

---

## 4. Ван де Граф — средневековая книжная сетка ⭐⭐⭐

**Суть:** Система полей, придуманная для рукописных книг. Поля не равные — они строятся по принципу 1:1.5:2:3 (внутреннее:верхнее:внешнее:нижнее). Текстовый блок при этом занимает ровно 4/9 страницы.

**Как реализовать:**
- Для страницы 1440 × 900px:
  - внутреннее поле (left): 160px
  - верхнее поле (top): 100px  
  - внешнее поле (right): 320px
  - нижнее поле (bottom): 200px
- Контентная зона: ~960 × 600px

**Где работает:** лонгриды, документальные проекты, «книжные» лендинги, brand identity сайты

**Эффект:** ощущение роскоши и академичности — как дорогой альбом или арт-буклет

---

## 5. Ratio Grid — сетка через соотношения сторон ⭐⭐

**Суть:** Вместо пиксельных размеров — всё строится через aspect-ratio. Блоки вложены друг в друга по принципу 16:9 → 4:3 → 1:1 → 3:4.

**Как реализовать:**
```css
.card-wide    { aspect-ratio: 16 / 9; }
.card-default { aspect-ratio: 4 / 3; }
.card-square  { aspect-ratio: 1 / 1; }
.card-tall    { aspect-ratio: 3 / 4; }
```
Сетка из карточек разных соотношений при общей ширине колонок создаёт визуальный ритм без хаоса.

**Где работает:** галереи, портфолио, медиа-сайты, bento grid

**Эффект:** редакционная верстка — как разворот в хорошем журнале

---

## 6. Optical Margin Alignment (оптическое выравнивание) ⭐⭐

**Суть:** Математически выровненный элемент выглядит «кривым». Дизайнер сдвигает его чуть за пределы формальной сетки — чтобы он *ощущался* ровным. Особенно актуально для знаков препинания, иконок и круглых форм.

**Как реализовать:**
```css
/* Кавычки «выезжают» за поле, чтобы текст выглядел ровным */
blockquote::before {
  margin-left: -0.5em;
}

/* Круглый аватар нужно сдвинуть чуть левее квадратного */
.avatar-round {
  margin-left: -4px;
}
```

**Где работает:** типографика, цитаты, иконки в навигации, логотипы в хедере

**Эффект:** разница между «собрано программистом» и «собрано дизайнером»

## 7. Diagonal Layout — диагональное зонирование ⭐⭐⭐

**Суть:** Разделение страницы идёт не горизонтально, а по диагонали. Одна зона «уходит» под углом, вторая заходит сверху. Создаёт движение и энергию.

**Как реализовать:**
```css
.section-diagonal {
  clip-path: polygon(0 0, 100% 0, 100% 85%, 0 100%);
  /* нижняя граница секции — диагональ, а не прямая */
}

.section-next {
  margin-top: -80px; /* следующая секция заходит под срез */
}
```

**Где работает:** tech-стартапы, спортивные бренды, агрессивный editorial

**Эффект:** страница «летит» — нет ощущения сложенных друг на друга прямоугольников

## 8. Typographic Grid — сетка, построенная от шрифта ⭐⭐

**Суть:** Вся компоновка строится от размера шрифта и межстрочного интервала. Высота блоков, отступы, размеры изображений — всё кратно `line-height` основного текста.

**Как реализовать:**
```css
:root {
  --base-font: 18px;
  --line-height: 28px;   /* это твоя единица всей системы */
}

.section     { padding: calc(var(--line-height) * 4) 0; }   /* 112px */
.card        { padding: calc(var(--line-height) * 1.5); }   /* 42px */
.image-block { height: calc(var(--line-height) * 12); }     /* 336px */
```

**Где работает:** контентные сайты, издательства, блоги высокого класса, design systems

**Эффект:** текст и визуал дышат в одном ритме — верстка выглядит «набранной», а не «накиданной»

## 9. Bento Grid с нарушенной симметрией ⭐⭐

**Суть:** Популярный bento-подход (как у Apple на страницах продуктов), но с намеренно асимметричным распределением блоков. Один крупный блок + несколько мелких — но не зеркально, а со смещением.

**Как реализовать:**
```css
.bento {
  display: grid;
  grid-template-columns: 2fr 1fr 1fr;
  grid-template-rows: auto;
}

.bento-hero {
  grid-column: 1;
  grid-row: 1 / 3;   /* занимает два ряда */
}

.bento-accent {
  grid-column: 2 / 4;  /* занимает две колонки */
}
```

**Где работает:** продуктовые лендинги, SaaS, feature-секции, about-страницы

**Эффект:** современно, технологично, дорого — без перегруза

## 10. Swiss Grid с цветовыми полями ⭐⭐⭐

**Суть:** Международный типографический стиль 1950-х. Строгая многоколонная сетка (12–16 колонок), минимум элементов, жёсткое выравнивание — но цветные поля, подложки и линии делают её экспрессивной.

**Как реализовать:**
- 12 равных колонок, gap 24px
- Контент размещается строго по сетке, никакого «по-глазу»
- Одно крупное цветовое поле уходит за пределы контейнера на всю ширину viewport
- Типографика: один гротескный шрифт, 3 кегля максимум

```css
.swiss-container {
  display: grid;
  grid-template-columns: repeat(12, 1fr);
  gap: 24px;
}

.full-bleed-bg {
  grid-column: 1 / -1;
  margin: 0 calc(-1 * var(--container-padding));
  background: #E63946;
}
```

**Где работает:** брендинг, культурные институции, архитекторы, дизайн-агентства

**Эффект:** абсолютная уверенность в себе — «я знаю что делаю» без лишних слов
