# 02. Авторизация

## Как работает

Codex CLI использует OAuth — вход через аккаунт ChatGPT в браузере. API-ключ не нужен.

## Требования

Активная подписка ChatGPT:
- **Plus** — $20/мес
- **Pro** — $200/мес
- **Business / Edu / Enterprise** — тоже поддерживаются

Подробнее: https://help.openai.com/en/articles/11369540-using-codex-with-your-chatgpt-plan

## Регион и VPN

OpenAI блокирует доступ с IP-адресов РФ, Беларуси и ряда других стран. Если ты в одной из них, `codex login` упадёт с ошибкой:

```
{"error":{"code":"unsupported_country_region_territory","message":"Country, region, or territory not supported"}}
```

Что делать:

1. Включи **системный VPN** (не браузерный) с выходом в США/ЕС/Японию.
2. Проверь, что трафик реально идёт через VPN:
   ```bash
   curl -s https://ipinfo.io/country
   ```
   Должно вернуть `US`, `DE`, `NL` и т.п. Если `RU` — VPN не поднялся или работает в split-tunnel без маршрутизации всего трафика.
3. Аккаунт ChatGPT тоже должен быть зарегистрирован не из РФ, иначе его заблокируют при первом входе.
4. После успешного логина `~/.codex/auth.json` сохранится локально. Периодически токен обновляется — в этот момент VPN снова понадобится. Проще держать VPN включённым постоянно.

## Что делает скрипт

1. Запускает `codex login`
2. Открывается браузер — входишь в аккаунт OpenAI
3. Токен сохраняется в `~/.codex/auth.json`

## Ручная авторизация

```bash
codex login
```

Или через device code (если браузер не открывается):

```bash
codex login --device-auth
```

## Проверка

```bash
codex   # если запускается без ошибок авторизации — всё ок
```

## Переавторизация

Если токен протух:

```bash
codex login
```
