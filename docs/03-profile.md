# 03. Профиль пользователя

## Зачем

Профиль помогает агенту подстроить стиль общения: уровень объяснений, язык, формат ответов. Без профиля агент отвечает «по умолчанию» — может быть слишком подробно или слишком коротко.

## Что делает скрипт

Задаёт 6 вопросов:

1. **Роль** — кем работаешь / чем занимаешься
2. **Опыт с кодом** — никогда / немного / регулярно
3. **Цели** — что хочешь создавать (боты, сайты, аналитика...)
4. **Сфера** — работа, учёба, хобби, фриланс, бизнес
5. **Стиль общения** — кратко / с объяснениями / пошагово
6. **Язык** — русский / English / оба

Ответы записываются в `~/.agents/AGENTS.md` в секцию `## User Profile`.

## Как изменить профиль

Открой файл и отредактируй:

```bash
open ~/.agents/AGENTS.md
```

Или в любом текстовом редакторе — найди секцию `## User Profile`.

## Где хранится

`~/.agents/AGENTS.md` — этот файл Codex CLI читает при каждом запуске как глобальные инструкции.
