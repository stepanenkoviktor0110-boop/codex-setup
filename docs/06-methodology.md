# 06. Методология (Skills + Agents)

## Что это

Набор инструкций, которые делают Codex CLI не просто чат-ботом, а полноценным AI-помощником для разработки. Включает:

- **Skills** (навыки) — пошаговые инструкции для конкретных задач
- **Agents** (агенты) — специализированные роли для проверки кода, тестирования, безопасности

## Что устанавливается

### Skills (34 навыка)

| Навык | Описание |
|---|---|
| `methodology` | Обзор методологии AI-разработки |
| `code-writing` | Написание кода с TDD и ревью |
| `code-reviewing` | Проверка качества кода |
| `new-user-spec` | Создание пользовательской спецификации |
| `new-tech-spec` | Создание технической спецификации |
| `decompose-tech-spec` | Декомпозиция на задачи |
| `do-task` | Выполнение задачи из файла |
| `do-feature` | Выполнение фичи целиком |
| `infrastructure-setup` | Настройка инфраструктуры проекта |
| `deploy-pipeline` | CI/CD и деплой |
| `security-auditor` | Аудит безопасности |
| `quick-learning` | Быстрый анализ и обучение |
| ... | и ещё 22 навыка |

### Agents (20 агентов)

Специализированные роли: code-reviewer, security-auditor, task-creator, test-reviewer, skeptic и другие.

## Где хранится

- `~/.agents/skills/` — навыки
- `~/.agents/agents/` — агенты
- `~/.agents/shared/` — общие ресурсы (шаблоны, референсы)
- `~/.agents/AGENTS.md` — глобальные правила работы

## Как использовать

Запусти Codex и используй команды навыков:

```
/new-user-spec    — создать спецификацию фичи
/new-tech-spec    — создать техспек
/do-task          — выполнить задачу
/do-feature       — выполнить фичу целиком
```

## Обновление методологии

```bash
cd ~/.agents && git pull origin master
```
