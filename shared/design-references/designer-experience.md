# Дизайнерский опыт — кросс-проектная база знаний

Глобальная база накопленного дизайнерского опыта, организованная по категориям проектов.
Используется скиллом design-retrospective для обучения на прошлых решениях.

**Формат:** append-only — новые записи добавляются в конец соответствующей подсекции.
Существующие записи не редактируются и не удаляются.

**Чтение:** агент ищет свою секцию по категории проекта через `## Category`.
