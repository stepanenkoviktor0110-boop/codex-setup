# Каталог бесплатных библиотек иконок и UI-элементов

Справочник библиотек для использования в дизайн-скиллах. Только ссылки и описания — ассеты не хранятся.

---

## Иконки

### Heroicons
- **URL:** https://heroicons.com
- **License:** MIT
- **Style:** Outline (24px) и Solid (20px), чистая геометрия. Созданы командой Tailwind CSS.
- **Use-cases:** Tailwind/React-проекты, административные панели, SaaS-интерфейсы. Идеальны при использовании Tailwind CSS.

### Lucide
- **URL:** https://lucide.dev
- **License:** ISC
- **Style:** Outline, 24x24, stroke-width 2px. Community-fork Feather Icons с расширенным набором.
- **Use-cases:** Универсальные проекты, замена Feather Icons с большим выбором. Хорошо подходят для dashboards и web-приложений.

### Phosphor Icons
- **URL:** https://phosphoricons.com
- **License:** MIT
- **Style:** 6 вариантов толщины (thin, light, regular, bold, fill, duotone). 1000+ иконок.
- **Use-cases:** Проекты, требующие гибкости стиля. Duotone-вариант для акцентных элементов. Мобильные и десктопные приложения.

### Tabler Icons
- **URL:** https://tabler.io/icons
- **License:** MIT
- **Style:** Outline, stroke-width 2px, 24x24. 5000+ иконок — один из крупнейших наборов.
- **Use-cases:** Проекты с большим количеством разнообразных иконок. Админ-панели, CRM, ERP-системы.

### Material Symbols
- **URL:** https://fonts.google.com/icons
- **License:** Apache 2.0
- **Style:** Variable font — настраиваемые weight, fill, grade, optical size. 3000+ иконок.
- **Use-cases:** Material Design проекты, Android-приложения, Google-экосистема. Проекты, требующие точного контроля над весом и заливкой.

### Radix Icons
- **URL:** https://www.radix-ui.com/icons
- **License:** MIT
- **Style:** Outline, 15x15, минималистичные. Созданы для Radix UI.
- **Use-cases:** Компактные интерфейсы, проекты на Radix UI/Shadcn. Подходят для мелких элементов (toolbar, sidebar).

### Simple Icons
- **URL:** https://simpleicons.org
- **License:** CC0 1.0
- **Style:** Монохромные логотипы брендов и сервисов. 3000+ брендов.
- **Use-cases:** Социальные ссылки, интеграции, технологические стеки, футеры с иконками брендов.

### Feather Icons
- **URL:** https://feathericons.com
- **License:** MIT
- **Style:** Outline, 24x24, stroke-width 2px. Минималистичные, 287 иконок.
- **Use-cases:** Небольшие проекты с ограниченным набором иконок. Простые интерфейсы, лендинги. Для расширенного набора — использовать Lucide.

## UI-элементы и компоненты

### Radix UI Primitives
- **URL:** https://www.radix-ui.com/primitives
- **License:** MIT
- **Style:** Unstyled accessible компоненты (dialog, dropdown, tooltip, popover и др.).
- **Use-cases:** Кастомные дизайн-системы, проекты с уникальным визуалом. Базовые примитивы для Shadcn UI.

### Shadcn UI
- **URL:** https://ui.shadcn.com
- **License:** MIT
- **Style:** Copy-paste компоненты на Radix + Tailwind CSS. Не npm-пакет — код копируется в проект.
- **Use-cases:** React/Next.js проекты с Tailwind. Быстрый старт с красивыми компонентами, полный контроль над кодом.

### Headless UI
- **URL:** https://headlessui.com
- **License:** MIT
- **Style:** Unstyled accessible компоненты от команды Tailwind CSS (menu, listbox, dialog, tabs).
- **Use-cases:** Tailwind CSS проекты, Vue и React. Когда нужна доступность без навязанных стилей.

## Иллюстрации

### unDraw
- **URL:** https://undraw.co/illustrations
- **License:** MIT-like (free, no attribution)
- **Style:** Flat SVG-иллюстрации с настраиваемым accent-цветом. Сцены из рабочей и повседневной жизни.
- **Use-cases:** Лендинги, onboarding-экраны, пустые состояния (empty states), маркетинговые страницы.

### Open Peeps
- **URL:** https://www.openpeeps.com
- **License:** CC0 1.0
- **Style:** Hand-drawn иллюстрации людей — конструктор из голов, тел, поз. Чёрно-белый штриховой стиль.
- **Use-cases:** Персонализированные иллюстрации, аватары, команды, about-страницы.

### Humaaans
- **URL:** https://www.humaaans.com
- **License:** CC0 (free plan) / paid
- **Style:** Модульные иллюстрации людей — mix & match элементы (причёски, одежда, позы).
- **Use-cases:** Презентации, лендинги, блог-посты, разнообразные персонажи.
